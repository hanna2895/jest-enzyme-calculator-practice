# TDD React

Built from this series of tutorials: [https://testdriven.io/tdd-with-react-jest-and-enzyme-part-one]

## Local Setup
```sh
$ git clone git@github.com:testdrivenio/react-calculator.git
```

```sh
$ cd react-calculator
```

```sh
$ yarn install
```

## Run Locally
```sh
$ yarn start
```

## Run Tests
```sh
$ yarn test
```
